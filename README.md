# SourcePoint Website

This is the GitHub Pages website for SourcePoint, hosting the privacy policy and support pages required for App Store submission.

## Live Site

Once published, the site will be available at:
- **Homepage**: https://nightlock305.github.io/sourcepoint-website/
- **Privacy Policy**: https://nightlock305.github.io/sourcepoint-website/privacy.html
- **Support**: https://nightlock305.github.io/sourcepoint-website/support.html

## Local Development

To view locally, simply open the HTML files in your browser:
```bash
open index.html
```

## Deployment

This site is automatically deployed via GitHub Pages. Any push to the `main` branch will update the live site.

## URLs for App Store

Use these URLs in your App Store Connect submission:
- **Privacy Policy**: https://nightlock305.github.io/sourcepoint-website/privacy.html
- **Support URL**: https://nightlock305.github.io/sourcepoint-website/support.html
